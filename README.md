
# Student Registration System

A full-stack web application for managing student registrations, built with PHP, MySQL, HTML, CSS, and JavaScript.

![Screenshot](screenshot.png) 

# Features

- **Student Registration Form** with real-time validation
- **Client-side validation** using JavaScript
- **Server-side validation** and SQL injection prevention using prepared statements
- **View All Students** page with formatted table
- **Responsive design** with clean UI
-**MySQL database** for data persistence

# Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** PHP (PDO for database connections)
- **Database:** MySQL
- **Server:** WampServer (local development)
- **Tools:** phpMyAdmin, Notepad++
